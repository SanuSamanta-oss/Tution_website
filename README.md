# BioLearn - Life Science & Biology Tuition Website

## Project Overview
- **Name**: BioLearn
- **Goal**: Professional website for a tuition teacher offering Life Science & Biology classes for WBBSE and CBSE board students
- **Teaching Medium**: Bengali & English

## Features

### Completed
- Hero section with animated background and statistics
- Dr. A.P.J Abdul Kalam quote section
- Class 10th Life Science courses (WBBSE & CBSE) with fees
- Class 12th Biology courses (WBBSE & CBSE) with fees
- Fee comparison table
- Class 5-9 Science Group section (fees decided after call)
- Distance Learning section (fees based on distance & board)
- Enrollment modal popup with phone number display
- WhatsApp & Direct Call buttons in enrollment modal
- Why Choose Us / About section
- Contact section
- Responsive design (mobile + desktop)
- Floating mobile enroll button
- Smooth scroll navigation with transparent-to-solid navbar
- Scroll animations on sections

### Fee Structure
| Course | WBBSE | CBSE |
|--------|-------|------|
| Class 10th Life Science | Rs. 500/month | Rs. 700/month |
| Class 12th Biology | Rs. 700/month | Rs. 900/month |
| Class 5-9 (All Science) | Decided after consultation | Decided after consultation |
| Distance Learning | Based on distance & board | Based on distance & board |
